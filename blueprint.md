# Blueprint

## Overview

This project is a Next.js application designed to be a starting point for building modern web applications. It includes a basic setup for a web server, a form component, and an API route for handling form submissions.

## Project Outline

### Styling and Design

*   **Global Styles:** `src/app/globals.css` provides the base styling for the application.
*   **Icons:** The project uses SVG icons located in `public/` and `src/assets/icons/`.
*   **Images:** The project uses SVG images located in `src/assets/images/`.

### Features

*   **Form:** A multi-step form is implemented using React components in `src/componentes/`.
*   **API Route:** A contact API route is defined in `src/app/api/contact/route.ts` to handle form submissions.
*   **ViewModels:** A `UseViewModels.ts` file is included in `src/viewmodels/` for managing the application's view logic.

## Current Plan

The current plan is to initialize a Git repository and publish the project to GitHub.
